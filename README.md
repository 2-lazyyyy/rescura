# Linyone

Linyone is a real-time, AI-assisted disaster relief and rescue coordination platform built for the ASEAN context. It connects survivors, volunteers, and organizations with shared situational awareness, live communication, and guided response tools during emergencies.

## About

Linyone prioritizes rapid coordination during crisis events. The platform combines live mapping, family safety tracking, and organization dashboards with a real-time data layer (Supabase and Ably) and an AI assistant (Gemini) to keep responders and communities aligned even when resources are constrained.

## Key Features

- Interactive rescue map with geotagged pins for incidents, hazards, and supply requests
- Real-time alerting and SOS broadcasts with live updates for connected clients
- Family safety check-ins, last-seen tracking, and configurable safety windows
- AI-assisted chat for survival guidance, first-aid, and triage support
- Volunteer and organization coordination dashboards with visibility into requests and activity
- Messaging and notification flows to keep responders and families informed

## Tech Stack

- Next.js 15 (React 19)
- Tailwind CSS v4 and Shadcn UI
- Supabase (PostgreSQL + Auth)
- Mapbox GL
- Ably
- Google Gemini AI
